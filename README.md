# Partners React

A simulation and replay tool for the Partners board game.

## Features

- **Simulation** – Run many games with configurable strategies (Random, Smart, Smart+)
- **Strategies** – Learn how strategies work, customize weights, and create custom presets
- **Replay** – Watch game replays with step-by-step history
- **Board Editor** – Customize board layout and colors

## Strategy documentation

See the **Strategies** page in the app for full documentation. You can:

- Read how Random, Smart, and Smart+ work
- Adjust scoring weights (progress, bump, lock, etc.)
- Save custom presets and use them in simulations
